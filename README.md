# 🐻🍉 BebrykBot

BebrykBot is a YouTube livestream chat bot with AI replies, commands, points, ranks, quests, duels, achievements and mini-events.

The bot was made for fun livestream interaction: viewers can collect watermelons, gain XP, complete stream quests, fight in duels, use shop items and talk with an AI-powered bot.

## Features

- YouTube live chat connection
- AI chat replies
- Custom bot personality
- Watermelon economy
- XP and rank system
- Viewer profile command
- Stream quests
- Mini-events
- Duels between viewers
- Duels with the bot
- Achievements
- Shop and inventory
- Stream streaks
- Donation command
- Owner/admin commands
- Optional Firebase support
